## Reasoning + Response
We need a 95% confidence interval for the population mean.

Given:
- σ = 17
- n = 420
- x̄ = 78
- z* = 1.96 for 95% CI

Standard error = σ/√n = 17/√420

√420 ≈ 20.4939

SE = 17/20.4939 ≈ 0.8295

Margin of error = 1.96 × 0.8295 ≈ 1.6258 ≈ 1.63

LCL = 78 - 1.63 = 76.37
UCL = 78 + 1.63 = 79.63

\boxed{(76.37, 79.63)}
