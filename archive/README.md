# archive/ — Organized Archive of Superseded Documents

Stale docs preserved with context. Not a junk drawer — organized by category.
Each doc maintained up to its deprecation point. May still contain gold.

## Categories
- `handoffs/` — old session handoff documents
- `strategy/` — superseded strategy docs (PIVOT, PLAN, etc.)
- `design/` — historical design docs (DESIGN.md, experiments.md)
- `research/` — completed research (prompt_engineering_research.md, papers.md)
- `session_logs/` — chronological project log
- `docs/` — superseded docs/ files
- `submissions_never_sent/` — CSVs prepared but never submitted to Kaggle
